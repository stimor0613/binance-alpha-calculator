港澳珠 3天2晚 H5行程卡

恒丰酒店｜轮渡去澳门｜威尼斯人寄存+小额赌场｜珠海早茶｜2026/5/29–5/31

香港段：4人 澳门段：2人 酒店：恒丰可吸烟双床房 赌场预算：1000–3000/人 带伞：必须
总览 天气 Day1 香港 Day2 澳门 赌场 Day3 珠海 规则 预算
总原则：少走路，不死排；吃饭排队超15–20分钟就换；发财车等超15分钟就打车；赌场22:00收手，最多缓冲到22:15；抽烟只在合法点，排队不抽。

一眼看懂路线

Day1：深圳/西九龙 → 恒丰酒店4人汇合 → 茶餐厅 → 房间网申港卡 → 旧油麻地警署 → 尖沙咀购物 → 天星小轮夜景 → 红伶饭店。

Day2：澳门2人小分队：恒丰 → 佐敦地铁到上环 → 金光飞航到氹仔 → 威尼斯人寄存 → 老城/大三巴/官也街 → 威尼斯人/伦敦人休整 → 20:00–22:00小额玩 → 关闸/青茂入珠海。

Day3：珠海早茶/早午茶 → 11:15–11:45出发深圳机场 → 16:50飞徐州。

最重要的5个阈值

  • 码头：09:10还没进闸，改下一班。
  • 发财车：等超15分钟，打车。
  • 寄存/办卡：45分钟没搞定，付费寄存或换伦敦人/巴黎人。
  • 餐厅：南屏20分钟、大利来15分钟、晚饭20分钟，超时就换。
  • 真人台：找低注超过10分钟，转电子百家乐。

烟民核心策略

  • 恒丰：优先回可吸烟房抽。
  • 香港外面：不在室内、有盖、排队、码头/地铁/巴士/的士站附近抽。
  • 澳门赌场：只去指定吸烟室。
  • 澳门街上：远离巴士站/的士站10米以上。

天气 & 带伞判断

结论：每人带轻便折叠伞。港澳珠深这几天闷热，港澳有阵雨/雷雨概率，珠海/深圳也可能热+局部雨。鞋子穿防滑、不怕水的。
